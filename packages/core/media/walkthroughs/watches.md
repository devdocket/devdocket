# Watch CI Pipelines and PRs

Watches are fire-and-forget monitoring for CI pipelines and pull requests. DevDocket notifies you in VS Code when a GitHub Actions or Azure Pipelines run completes or fails, or when a pull request is merged.

## Add a Watch from a Work Item

If you have a provider-linked item in **Focus** or **Queue**:
1. Right-click the item
2. Select **Watch CI**
3. DevDocket starts monitoring the related pipeline or pull request for you

## Add a Watch Manually

You can also add a watch manually:
- Open the Command Palette and run **DevDocket: Watch Pipeline Run** or **DevDocket: Watch Pull Request**
- Or use the button below to start watching a pipeline run now

---

[Watch a Pipeline Run](command:devdocket.watchRun)
